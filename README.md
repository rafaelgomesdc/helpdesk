Pessoa 1 — Controle de Usuários

Responsável por:

autenticação;
login/logout;
recuperação de senha;
perfis de acesso;
permissões e regras de autorização;
segurança básica (hash de senha, sessões, tokens etc.).
Entregas
Tela de login;
Middleware/autorização;
CRUD de perfis;
Controle de permissões por usuário/perfil.
Pessoa 2 — Cadastro de Usuários

Responsável por:

cadastro de usuários;
setores;
cargos;
informações de contato;
edição/exclusão;
validações de formulário.
Entregas
CRUD de usuários;
CRUD de setores;
CRUD de cargos;
Tela de detalhes do usuário;
Campos de contato (telefone, email, endereço etc.).
